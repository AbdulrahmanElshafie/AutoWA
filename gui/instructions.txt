Communication System Usage Instructions:

Concepts:
- Pause => For emergency use only to pause the program. When pressed, the program sends the current permit then saves progress and stops.
- Execute => Executes the program and sends the permits.
- Restart => Starts sending permits from the beginning.
- Cancel => Closes the program.
- Update Sheet => Reads the input file and updates the data to be sent.
- Batch => Number of permits per single WhatsApp account.
- Round => Total permits sent across all accounts = accounts_count * batch_size.
- Timing Profile => A profile containing the time taken while typing between words to simulate a normal person's typing.
==========================================================
After preparing what you want to send (travel permits, 53 filling, communication messages), move the data to the Jobs Data File (CSV) and place the data in the designated columns.
= In Permits:
Place the phone number, officer name, and national ID.
= In 53 Filling:
Place the phone number and national ID.
= In Communication Messages:
Place the phone number, officer name, and the message text.
==========================================================
Sending Steps:
1- Select all required paths and confirm them (Permits folder, input CSV file).
2- Select the desired Timing Profile - these are the wait times between words during typing.
3- Choose the operation mode, set minimum and maximum wait between messages and batches, and set the batch size.
4- Update the data then press Execute.
5- It is very important not to move the mouse while the program is running to avoid problems.
